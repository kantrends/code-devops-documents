# Pull Request Notification

## Use watch setting in the repo:

- Click on `watch` icon in the repository for which you wanted to get notification whenever a pull request is created.

  ![image](https://github.com/user-attachments/assets/ecd05c7c-4e19-4a7d-bf13-38e6cb20b212)

- Select the `Custom` option.

  ![image](https://github.com/user-attachments/assets/26084d44-d952-4d92-9f5f-38b00175d675)

- Select `Pull Request` option and apply it.

  ![image](https://github.com/user-attachments/assets/a210ed64-b430-4209-9fef-914ca0adb445)

- Once done refresh the page and you will see the `watch` button will change to `Unwatch 0`

  ![image](https://github.com/user-attachments/assets/bc9f9e7f-8f91-4b92-ade3-cd99c256a80d)

- By doing these steps you will receive an email by github whenever a pull request is opened, closed or repoened.

## If you want to receive an email whenever you create a pull request you need to do some setting:

- Go to [Github Notification Setting](https://github.com/settings/notifications)
- In `Subscription` tab click on ` Customize email updates` and select `Include your own updates` and save it.

  ![image](https://github.com/user-attachments/assets/713edb7d-d768-4c53-abda-cd549c456317)

- After this setting you will receive email for the pull request which you create as well as other people who are watching this will get the email for pull request.

## Troubleshoot step if you are not receiving email notification after all these steps:

- Go to [Github Notification Setting](https://github.com/settings/notifications)
- Confirm if the default email is yours.

  ![image](https://github.com/user-attachments/assets/2a89c5b8-e237-4822-90b4-d2c7b21f5c5a)

- In `Subscriptions` check if you have selected email option in watch or not.

  ![image](https://github.com/user-attachments/assets/69ac1228-7cc0-45af-8160-47e71a229d4d)

- If you have already selected it. Please deselect the email option, Save it. Then select the email option and save it.
- Also check if the repo is not in ignored repositories which is the last option in `Subscriptions` tab.

  ![image](https://github.com/user-attachments/assets/7afb947b-71c5-4c44-8d8c-23db7c245ef4).

- By doing these steps you will get email each time whenever a `Pull Request` is created.
