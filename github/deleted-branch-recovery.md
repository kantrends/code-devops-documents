# Document for recovering deleted branch from github repositories.
 If you have deleted a branch from your repo and wanted to recover it. You can follow the below mentioned steps to get it back.

## Step 1:

- Open your repository and select the Activity from the side panel.

 ![Activity-panel.png](./resources/Activity-panel.png)

- You can see the deleted branch activity on the top. Just click on the 3 dots on the extreme right of that activity.

![three-dot-for-activity-panel.png](./resources/three-dot-for-activity-panel.png)

- Select the Restore Branch.

![restore.png](./resources/restore.png)

- After that you can see the deleted branch back in the branch tree.
