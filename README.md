## DevOps Templates

### Build

**Simplify your build process with our curated templates:**

- **Maven Project**: Utilize the [Maven template](./azure_devops/pipeline/examples/01.%20maven-dtr-ucp) for building Java applications.
- **Node.js Project**: Streamline your Node.js application builds with the [Node.js template](./azure_devops/pipeline/examples/03.%20nodejs-native-app-service).
- **.NET Project**: Accelerate your .NET application development using the [.NET template](#).
- **Python**: Build your native python apps with [python template](./azure_devops/pipeline/examples/04.%20python-native-app-service)
- **Docker Builds**: Facilitate containerization with our [Docker build templates](#).

### Deploy

**Deploy your applications seamlessly across various platforms:**

- **MKE (Mirantis Kubernetes Engine)**: Deploy to MKE using this [template](#).
- **ACI (Azure Container Instances)**: Leverage ACI for container deployment with this [guide](./azure_devops/pipeline/examples/06.%20aci).
- **Azure Apps**: Utilize [this template](./azure_devops/pipeline/examples/02.%20maven-acr-azure-app-service) for deploying to Azure Apps.
- **AKS (Azure Kubernetes Service)**: Deploy to AKS by following [these instructions](#).
- **VM (Virtual Machine)**: Deploy your applications to a VM with [this template](./azure_devops/pipeline/examples/09.%20linux-vm-deploy-jar-using-ansible).
- **Deploy DACPAC to SQL Server**: Use [this template]() for deploying database changes with DACPAC files to SQL Server.
- **Liquibase Database Versioning**: Automate and manage your database schema changes with [this template](./azure_devops/pipeline/examples/07.%20liquibase).
- **Restart Azure App Services**: To restart Azure App Services follow [this template](./azure_devops/pipeline/examples/restart-azure-app-services)
